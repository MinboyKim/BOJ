## 제출 방법
1. repository를 fork한 뒤 clone합니다.
2. fetch/pull 을 이용하여 저장소를 업데이트합니다.
3. 문제를 풀고 소스코드를 작성합니다. 파일명은 `난이도_문제번호_handle` 로 합니다. ex) GOLD4_1707_Minboykim.cpp
4. commit 후 push합니다. commit 명은 `[handle] [날짜] [문제번호_난이도] submit` 로 합니다. 만약 푼 문제가 여러 문제일 경우 문제 번호를 콤마 (,) 로 구분합니다. ex)Minboykim 22/11/17 1259_BRONZE5, 1654_SILVER2 submit 
6. PR을 보낸 후 code review를 진행합니다. PR 제목은 commit 명과 동일합니다.
7. 모든 구성원들의 code review가 끝나면 마지막으로 code review를 진행한 사람이 merge 합니다.
